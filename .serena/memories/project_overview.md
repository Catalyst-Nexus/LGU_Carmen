# Animal Farm System Project Overview

## Purpose
A modern React + TypeScript admin system template with authentication, role-based access control (RBAC), dynamic image uploads, and multiple management modules for an animal farm system.

## Key Features
- Authentication system with secure login and registration
- Dashboard with stats and navigation
- Role-Based Access Control (RBAC) management
- Assignment/task tracking
- Module management (enable/disable features)
- User management and activation
- Dynamic logo and profile picture uploads via Supabase Storage
- User profile management

## Tech Stack
- **Frontend Framework**: React 19
- **Language**: TypeScript
- **Build Tool**: Vite (with Tailwind CSS)
- **Routing**: React Router v7
- **State Management**: Zustand, Redux Toolkit
- **Styling**: Tailwind CSS (using @tailwindcss/vite)
- **UI Components**: Radix UI, shadcn/ui components
- **Backend/Storage**: Supabase (for authentication and image storage)
- **Icons**: Lucide React, Tabler Icons, React Icons
- **Table Library**: TanStack React Table
- **Testing**: Vitest with React Testing Library
